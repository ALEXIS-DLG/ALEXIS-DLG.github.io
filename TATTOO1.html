"CUIDADOS ANTES DE REALIZAR UN TATUAJE"

HIGIENE Y ESTERILIZACION

  1. Lavado de manos antes y después de cada sesión.
  2. Uso de guantes desechables (cambiar cada vez que se toque algo no estéril).
  3. Esterilización del equipo (autoclave para elementos reutilizables o uso de material desechable).
  4. Limpieza del área de trabajo con desinfectantes apropiados.
  5. Uso de agujas y tubos desechables o debidamente esterilizados.
  6. Tintas envasadas individualmente y no reutilizar restos.
    7.

PREPARACION DEL CLIENTE

  1. Evaluar la salud del cliente: Preguntar sobre alergias, enfermedades de la piel, problemas de coagulación, etc.
  2. Consentimiento informado: Hacer firmar un documento que explique riesgos y cuidados posteriores.
  3. Rasurado y limpieza: Rasurar la zona a tatuar con una máquina estéril y desinfectarla con alcohol u otro antiséptico.
    4.

PREPARACION DEL DISEÑO Y MATERIALES

  1. Stencil o boceto claro: Asegurarse de que el cliente apruebe el diseño y su ubicación.
  2. Tintas seguras: Usar pigmentos homologados y específicos para tatuajes, nunca tintas industriales.
  3. Recipientes para tinta: Usar vasitos desechables y no reutilizarlos.
    4.

CONDICIONES DEL AREA DE TRABAJO

  1. Área limpia y delimitada: Separar el área de tatuado del resto del local (recepción, aseo, etc.).
  2. Iluminación adecuada: Para asegurar precisión y visibilidad.
  3. Ventilación y orden: Evitar ambientes cargados o con polvo.
    4.

CONDICIONES DEL AREA DE TRABAJO

  1. Explicar el proceso: Informar al cliente paso a paso para que se sienta seguro.
  2. Respetar los tiempos: No tatuar apresuradamente ni prolongar innecesariamente la sesión.
  3. No tatuar bajo ciertas condiciones: Rechazar tatuar a personas bajo efectos de alcohol/drogas, embarazadas (sin autorización médica), menores sin consentimiento, o con infecciones activas.
    4.

"CUIDADOS DESPUES DE REALIZAR UN TATUAJE"

LIMPIEZA Y DESINFECCION DEL AREA DE TRABAJO

  1. Desechar correctamente el material contaminado: Agujas, guantes, vasitos de tinta y cualquier elemento desechable usado.
  2. Limpiar la máquina y superficies: Desinfectar todo con productos adecuados (por ejemplo, solución con clorhexidina o alcohol isopropílico).
  3. Cambiar fundas protectoras: Retirar y desechar fundas plásticas de cables, grip, etc., y colocar nuevas para la próxima sesión.
  4. Esterilizar material reutilizable: Si hay piezas no desechables, esterilizarlas en autoclave.
    5.

CUIDADO DEL TATUAJE RECIEN HECHO (EN EL CLIENTE)

  1. Limpieza del tatuaje: Limpiar con solución salina o jabón antibacteriano.
  2. Aplicar pomada cicatrizante: (como bepantol, bacitracina, etc., según lo recomendado).
  3. Cubrir el tatuaje: Con film plástico o vendaje estéril, explicando al cliente cuánto tiempo debe mantenerlo (esto puede variar según la técnica y el tipo de vendaje usado).
    4.

INSTRUCCIONES AL CLIENTE (POSTRATAMIENTO)

  1. Cómo lavar y cuidar el tatuaje en casa (agua tibia, jabón neutro, secado suave, aplicar pomada).
  2. Qué evitar: exposición al sol, piscina, sauna, rascarse o retirar costras.
  3. Signos de alarma: enrojecimiento excesivo, pus, fiebre o dolor intenso (recomendar acudir al médico si aparecen).
    4.

REGISTRO Y DOCUMENTACION

  1. Guardar el consentimiento firmado.
  2. Registrar datos del tatuaje: fecha, tipo de diseño, ubicación en el cuerpo, reacciones del cliente, pigmentos usados (esto puede ser requerido por normativas sanitarias).
    3.

AUTOCUIDADO DEL TATUADOR

  1. Lavar manos nuevamente después de terminar.
  2. Evaluar el uso de EPP (equipo de protección personal): para futuras mejoras o reemplazos si hubo alguna exposición accidental.
  3. Descanso físico: especialmente si ha sido una sesión larga o exigente.
    4.

"MATERIALES PRIMORDIALES"

EQUIPOS BASICOS

  1. Máquina de tatuar (rotativa o de bobinas).
  2. Fuente de poder (con regulador de voltaje).
  3. Cable clip cord / RCA (dependiendo del tipo de máquina).
  4. Pedal (foot switch).
    5.

PREPARACION DEL AREA A TATUAR

  1. Rasuradora desechable.
  2. Jabón antiséptico o verde (green soap).
  3. Toallas de papel (tipo Scott o Wypall).
  4. Botella aplicadora (para diluir y aplicar jabón).
  5. Guantes de nitrilo o látex (sin talco).
    6.

HIGIENE Y PROTECCION

  1. Film plástico o dermofilm (para cubrir zonas).
  2. Bolsas o fundas desechables (para cables, máquinas, botellas).
  3. Bandejas o superficies protegidas.
  4. Desinfectantes de superficies (alcohol isopropílico, clorhexidina).
  5. Autoclave o esterilizador (si se usan piezas reutilizables).
    6.

MATERIAL DE TATUADO

  1. Agujas estériles (liners, shaders, magnums, según el trabajo).
  2. Grip desechable o esterilizable.
  3. Tinta para tatuajes (de calidad certificada y específicas para uso humano) .
  4. Vasos pequeños (ink caps) para colocar la tinta.
  5. Vaselina o ungüento lubricante (para facilitar el trabajo y proteger la piel).
  6. Bastoncillos de algodón y paletas de madera (opcional, para aplicar producto).
    7.

DISEÑO Y APLICACION

  1. Papel hectográfico o transfer paper.
  2. Impresora térmica (para transferir el diseño).
  3. Solución para stencil (Stencil Stuff, Electrum, etc.).
  4. Marcadores quirúrgicos o de piel (para detalles manuales).
    5.

CUIDADOS POSTERIORES

  1. Film plástico o vendaje estéril (para cubrir el tatuaje)r.
  2. Pomada cicatrizante (recomendada por el tatuador).
  3. Instrucciones impresas de cuidados post-tatuaje.
    4.

DOCUMENTACION Y REGISTRO

  1. Consentimiento informado (formulario para firmar).
  2. Ficha del cliente (registro de fecha, tintas, zona tatuada).
  3. Identificación del lote y marca de tintas/aguja (según normativa sanitaria).
    4.
